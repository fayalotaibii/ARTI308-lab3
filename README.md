# Deepfake Voice Recognition - EDA

### Project Overview
This project performs an Exploratory Data Analysis (EDA) on the **DEEP-VOICE** dataset. The goal is to identify key acoustic features that differentiate authentic human speech from AI-generated (Deepfake) voice clones.

### Key Analysis Performed:
- **Univariate Analysis:** Distribution of spectral features.
- **Bivariate Analysis:** Comparing Real vs. Fake audio characteristics.
- **Multivariate Analysis:** Correlation matrix of 26 acoustic features.

### Dataset
The dataset contains features extracted from one-second audio segments of eight public figures, processed using Retrieval-based Voice Conversion (RVC).
